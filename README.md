# Cookie

Mini-jogo de clique com recorde guardado em cookie.

## Arquivos

- `index.html`: layout do jogo e área do alvo.
- `style.css`: estilo da página.
- `script.js`: lógica de jogo, tempo, pontuação e cookie de recorde.

## Funcionamento

1. Clique em `Começar`.
2. Clique no alvo que aparece na área dentro de 10 segundos.
3. O JavaScript compara a pontuação atual com o recorde armazenado no cookie.
4. Se você superar o recorde, o cookie é atualizado e a nova melhor pontuação fica gravada no navegador.
